# less-os

简介：<br />
less-os是一个简单实用的less库，由作者从日常项目中积累整理而来，包含了浏览器兼容前缀补全和IE8等低版本浏览器兼容，适用于移动端和PC端开发.

<br /><br />
文件说明：<br />
less_os.less   -less库主文件，存放公共属性和方法
whir_css.less  -less源码，存放项目less源码
